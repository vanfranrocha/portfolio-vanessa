# Portfolio Vanessa Rocha

- `case-btg.html`: Case Study Banco BTG
- `case-jotaja.html`: Jotaja

- `case-datapage.html`: datapage
- `case-compare-compre.html`: Compare e Compre

## Publicacao

1. Importe este repositorio na Vercel.
2. Mantenha o preset como `Other`.
3. Nao e necessario configurar build command ou output directory.

Para visualizar localmente:

```bash
python3 -m http.server 4173
```
